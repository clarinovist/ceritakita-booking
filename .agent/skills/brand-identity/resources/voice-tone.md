# Copywriting: Voice & Tone Guidelines

When generating text, adhere to this brand persona.

## Brand Personality Keywords
* Warm & Approachable
* Artistic & Storytelling
* Elegant & Professional
* Empathetic
* Nostalgic yet Modern

## Grammar & Mechanics rules
* **Headings:** Use Title Case for main headings. Elegant serifs should be used for headlines.
* **Punctuation:** Soften the tone. Avoid being overly clinical. Use evocative language for descriptions (e.g., "Abadikan momen Anda" instead of "Buat pesanan").
* **Clarity:** Maintain simplicity, but allow for artistic flair in descriptions.

## Language
* Primary: Indonesian (Bahasa Indonesia) for user interface, following the "CeritaKita" (Our Story) theme.

## Terminology Guide

| Do Not Use | Use Instead |
| :--- | :--- |
| "Customer" | "Klien" / "Sahabat Cerita" |
| "Order" | "Sesi Foto" / "Momen" |
| "Dashboard" | "Beranda Studio" |
| "Submit" | "Simpan Momen" |
